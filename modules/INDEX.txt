================================================================================
                        📊 MODULE DESCRIPTIONS
================================================================================

01_DOCCHECK - Free Assessment Tool
   Purpose: Identify missing documents and recommend service tier
   Users: Sellers (unauthenticated)
   Flow: 8 questions → Document matrix → Tier recommendation
   Key Feature: Email capture for lead generation

02_DOCREADY - Order & Payment Processing
   Purpose: Handle tier selection, Stripe payment, account creation
   Users: Sellers
   Flow: Tier selection → Stripe checkout → Auto account creation → OCR
   Key Feature: Automatic account setup post-payment

03_DOCUMENTS - Secure Document Management
   Purpose: Store, track, verify documents through their lifecycle
   Users: Sellers, Operators
   Flow: Upload → OCR verify → Status track → Expire monitor
   Key Feature: Private storage + expiry alerts

04_SMARTCMA - Price Intelligence Reports
   Purpose: Provide data-backed pricing and market insights
   Users: Sellers (post-completion)
   Flow: Trigger → PropCheck valuation → Comparables → PDF report
   Key Feature: Market-specific pricing with confidence score

05_OPERATOR - Operations Queue Dashboard
   Purpose: Manage document procurement workflow
   Users: Fizbo operators (admin only)
   Flow: View queue → Update status → Coordinate suppliers → Track completion
   Key Feature: Real-time order queue with supplier management

06_PAYMENTS - Stripe Integration & Refunds
   Purpose: Process payments and manage refund policy
   Users: Sellers (payment phase)
   Flow: Price calculation → Stripe checkout → Webhook → Refund processing
   Key Feature: Dynamic pricing with energy cert discount

================================================================================
                          🎯 KEY BUSINESS RULES
================================================================================

DOCUMENT VALIDITY:
  • Caderneta Predial: 12 months
  • Certidão Permanente: 6 months
  • Certificado Energético: 10 years
  • Licença Utilização: No expiry
  • Habilitação Herdeiros: Permanent

PRICING TIERS:
  • Standard: €399 (15–20 days, urban resident)
  • Premium: €899 (20–30 days, non-resident/inherited)
  • DocExpress: €1,499 (10 days urgent)
  • Custom: Quote (rural/complex)

REFUND POLICY:
  • 100%: If no docs requested in 5 working days
  • 50%: If work started but no docs obtained
  • 0%: Once any document obtained

DOCUMENT STATES:
  1. Missing (required, not yet requested)
  2. Requested (operator asked for it)
  3. Pending (being processed at supplier)
  4. Complete (received & verified)
  5. Expiring (warning period triggered)
  6. Not Required (logic determined not needed)

                          📖 HOW TO VIEW DIAGRAMS
================================================================================

OPTION 1: VS Code (RECOMMENDED)
  1. Install "PlantUML" extension from VS Code marketplace
  2. Open any .puml file
  3. Click preview button (top right) or press Alt+D

OPTION 2: Online
  1. Visit: https://www.plantuml.com/plantuml/uml/
  2. Copy-paste .puml file contents
  3. View rendered diagram instantly

OPTION 3: Command Line
  1. Install: plantuml (see VIEWING_GUIDE.md)
  2. Run: plantuml modules/ -r -tpng
  3. PNG files generated in each folder
